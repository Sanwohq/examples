# Sanwo Flutter Example

A minimal Flutter app demonstrating the `sanwo_flutter` SDK with all supported payment providers.

## Setup

1. Replace the placeholder public keys in `lib/main.dart` with your actual test keys from each provider dashboard.

2. Install dependencies and run:

```bash
flutter pub get
flutter run
```

## Providers

- **Paystack** — card, bank transfer, mobile money
- **Flutterwave** — card, bank, USSD, mobile money
- **Razorpay** — card, UPI, netbanking (INR)
- **Monnify** — card, bank transfer
- **Interswitch** — Webpay checkout
