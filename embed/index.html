Sanwo Embed

Zero-code payment buttons — just add data attributes. No build tools, no frameworks, no JavaScript required.

1. Zero-Code Button

Just HTML data attributes — no JavaScript needed. Click to pay with Paystack.

<button data-sanwo-provider="paystack" data-sanwo-key="pk_test_xxxxx" data-sanwo-amount="500000" data-sanwo-currency="NGN" data-sanwo-email="customer@example.com" > Pay ₦5,000 </button>

2. Flutterwave Button

Same pattern, different provider — just change the data attributes.

3. Programmatic Usage

For more control, use the JavaScript API.

const sanwo = Sanwo.create({ provider: 'paystack', publicKey: 'pk_test_xxxxx' }); const result = await sanwo.checkout({ amount: 5000, currency: 'USD', customer: { email: 'user@example.com' } }); if (result.status === 'successful') { alert('Paid! Ref: ' + result.reference); }
